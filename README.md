
# ExU-Atlas
The goal of this project is to determine the optimal execution unit topology-fpga pair for various use cases, such as situations where performance is the goal, or when cost is the goal, or when a balance is preferred.
The execution unit topologies are written in Sysverilog and are tested for correctness using UVM Testbenches. Their speeds are evaluated using propagation delay measurements and their costs are measured using logic element statistics.
